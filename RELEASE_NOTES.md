# Dental Ark 0.1.2

**Dental Ark Community Edition is free to download and use.** This release keeps the core desktop workflow available without a subscription requirement.

## What stands out in this release

- Local SQLite clinic records keep patients, appointments, visits, treatment notes, and follow-up state on the clinic machine.
- The front-desk flow connects appointments, arrival, waiting, treatment, checkout, balances, and follow-up in one daily queue.
- Native FDI charting links conditions, tooth-level notes, images, and visit history while preserving draft, confirmed, and amended audit states.
- Itemized billing records cash, card, and transfer payments, adjustments, outstanding balances, and clinic-ledger review.
- Manual backup produces a restorable ZIP containing the database, images, configuration, and manifest.
- Community Edition supports up to 50 patients, 200 visits, 200 appointments, and 100 images on the published Windows and Linux packages.

## Choose the right package

- **Linux x64 (APPIMAGE):** `dental-ark-0.1.2-linux-x64.AppImage` — Portable Linux desktop package
- **Linux x64 (DEB):** `dental-ark-0.1.2-linux-x64.deb` — Debian, Ubuntu, Mint, and compatible systems
- **Linux x64 (RPM):** `dental-ark-0.1.2-linux-x64.rpm` — Fedora, RHEL, openSUSE, and compatible systems
- **Windows x64:** `dental-ark-0.1.2-windows-x64-setup.exe` — Guided Windows installer

## Before you install

- Patient records, clinical notes, images, billing entries, and backups stay on the clinic machine; never attach patient data to a public issue.
- Keep the installer filename and checksum together when handing a package to another machine.
- Read the [help center](https://hannes-software.com/dental-ark/help/) for supported inputs, workflow boundaries, and troubleshooting.
- Optional licensed features are described on the website without changing the free Community Edition download.
