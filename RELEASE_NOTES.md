# Dental Ark 0.1.3

**Dental Ark Community Edition is free to download and use.** This release keeps the core desktop workflow available without a subscription requirement.

## What stands out in this release

- Added a unified appointment composer for new bookings and rescheduling, including patient, clinician, chair, service, date, and time context plus non-blocking conflict previews backed by authoritative local backend validation.
- Appointment procedure items now carry into check-in as proposed visit items, preserving source identity through the visit and billing workflow.
- Refined responsive patient, schedule, finance, and data-safety workbenches; a fresh workspace no longer surfaces a false backup emergency.

## Choose the right package

- **Linux x64 (APPIMAGE):** `dental-ark-0.1.3-linux-x64.AppImage` — Portable Linux desktop package
- **Linux x64 (DEB):** `dental-ark-0.1.3-linux-x64.deb` — Debian, Ubuntu, Mint, and compatible systems
- **Linux x64 (RPM):** `dental-ark-0.1.3-linux-x64.rpm` — Fedora, RHEL, openSUSE, and compatible systems
- **Windows x64:** `dental-ark-0.1.3-windows-x64-setup.exe` — Guided Windows installer

## Before you install

- Patient records, clinical notes, images, billing entries, and backups stay on the clinic machine; never attach patient data to a public issue.
- Keep the installer filename and checksum together when handing a package to another machine.
- Read the [help center](https://hannes-software.com/dental-ark/help/) for supported inputs, workflow boundaries, and troubleshooting.
- Optional licensed features are described on the website without changing the free Community Edition download.
